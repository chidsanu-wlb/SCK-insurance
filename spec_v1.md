4. **วันเริ่มสัญญาฉบับปัจจุบัน** = วันสิ้นสุดความคุ้มครอง - 1 ปี
5. **Date Format** = วัน เดือน ปี (ใช้ปี พ.ศ. โดยตรง ไม่ต้องแปลง -543)

    ## 2. กฎเกณฑ์เรื่องอายุ (และเงื่อนไขการตรวจสอบกรณีอายุเกินเกณฑ์)

    > **เกณฑ์:** อายุต้องไม่เกิน 15 ปีบริบูรณ์ ณ วันต่อสัญญาฉบับใหม่

    - **กรณีปกติ:** วันเกิดบุตรกรณีปกติ - สุ่มในช่วง `(7300 - 10950)` วัน
- **กรณีอายุเกินเกณฑ์:** มีอายุเกิน 65 ปีบริบูรณ์ ณ วันต่อสัญญาฉบับใหม่
  - สุ่มวันเกิดในช่วง: `(15 สิงหาคม 2503 + จำนวนวันที่ต่ออายุล่วงหน้า + 1)` ถึง `15 สิงหาคม 2504` (อายุ > 65 ปี)

## Workflow

1. อ่าน input CSV 2. อ่าน column **"Timeline ของประกันที่จะทำการต่ออายุ"** เพื่อหาจำนวนวันที่จะต่ออายุล่วงหน้า 3. อ่าน column **"Bundle"** 4. อ่าน column **"Family"** เพื่อสร้างสมาชิกตามเงื่อนไข
   ทุกกรณีต้องสร้างข้อมูลสมาชิกตามที่ระบุ
   ถ้าเป็นกรณี "เสียชีวิต" → ใส่สถานะ "เสียชีวิต"
   ถ้าเป็นกรณี "อายุเกิน" → สร้างสมาชิกพร้อมวันเกิดที่ทำให้อายุเกินเกณฑ์
   ต้องสร้างครบตามจำนวนที่ระบุ (คู่สมรส, ลูก 1, ลูก 2)
